### Project Description

This is a practise app made with reactjs and firebase.


### Features

- Live store updates
- Facebook, github and twitter login
- Database stored in firebase


### Dependencies

* React

### How to run

* Clone the repo
* Run "npm install" to install dependencies
* Run "npm start" to start the server
